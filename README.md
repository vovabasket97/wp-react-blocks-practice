## WP React Blocks

This repository was created for practicing/testing some features with WordPress react blocks.

## Package

### Installation

Install npm dependencies

```bash
npm install
```
    
### Run Locally


Run frontend project assembler

```bash
npm run dev
```

To build frontend project manually

```bash
npm run build
```
